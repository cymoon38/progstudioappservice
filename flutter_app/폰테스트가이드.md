# 실제 폰에서 테스트하기 가이드

## Android 폰 준비
1. **개발자 옵션 활성화**
   - 설정 → 휴대전화 정보 → "빌드 번호"를 7번 연속 터치
   
2. **USB 디버깅 활성화**
   - 설정 → 개발자 옵션 → "USB 디버깅" 활성화

3. **USB로 폰 연결**
   - USB 케이블로 PC와 폰 연결
   - 폰에서 "USB 디버깅 허용" 팝업이 뜨면 "허용" 선택

## 연결 확인
터미널에서 다음 명령어 실행:
```bash
flutter devices
```

## 앱 실행
폰이 연결되면 다음 명령어로 앱 실행:
```bash
cd flutter_app
flutter run
```

## 와이파이로 연결 (선택사항)
USB 없이 와이파이로도 연결 가능합니다:

1. **첫 연결 시 USB 필요**
   - USB로 한 번 연결 후:
   ```bash
   adb tcpip 5555
   ```

2. **폰의 IP 주소 확인**
   - 설정 → 휴대전화 정보 → 상태 → IP 주소 확인
   - 또는: 설정 → Wi-Fi → 연결된 네트워크 → IP 주소 확인

3. **와이파이로 연결**
   ```bash
   adb connect [폰의IP주소]:5555
   ```
   예: `adb connect 192.168.0.100:5555`

4. **연결 확인**
   ```bash
   flutter devices
   ```

## 문제 해결
- 폰이 인식되지 않으면:
  1. USB 케이블 다시 연결
  2. USB 디버깅 다시 활성화
  3. PC에서 ADB 드라이버 재설치 (필요시)
  4. `adb kill-server` 후 `adb start-server` 실행
