가격:30코인
기능:아이템을 사용하고 싶은 게시물을 선택하고 사용하면 사용한 게시물이 피드 상단에 위치하게 됨. 당근의 끌올이랑 비슷한 기능. 시간이 지나 다른 게시물 때문에 피드 아래로 내려가도 해당 아이템을 사용하면 피드 상단에 다시 위치했다가 다른 게시물이 올라오면 기존처럼 아래로 한칸씩 내려감

아이템명:목탄
가격:100코인
기능:장작 아이템과 똑같이 아이템을 사용하면 게시물이 상단에 위치. 장작과 달리 사용한 게시물은 상단에 1시간동안 고정된다. 사용 1시간 후 장작을 사용한 게시물과 똑같이 새 게시물에 밀리게됨.
또한 사용한 게시물 오른쪽 상단 인기작품 불꽃모양 아이콘 표시 위치에 100%채택 문구가 표시되며 이 게시물이 인기작품으로 채택되면 불꽃모양은 문구 아래에 표시됨.
아이템 사용후 24시간이 지나고 사용자가 앱에 접속하면 댓글 채택하기 창이 뜨며 게시물 작성자가 선택한 댓글 작성자 한명이 50코인에 채택됨. 이 채택은 채택 제한 수인 3개에 포함되지 않음. 아이템 사용후 48시간이 지나도 사용자가 댓글 채택을 안할 경우 랜덤 채택.
