# 모바일에서 접속하는 방법

## 서버 시작하기

1. **start-server.bat** 파일을 더블클릭하여 서버를 시작하세요.

2. 서버가 시작되면 다음과 같은 메시지가 표시됩니다:
   ```
   서버 주소: http://192.168.219.105:8000
   또는: http://localhost:8000
   ```

## 모바일에서 접속하기

### 필수 조건
- ✅ PC와 모바일이 **같은 Wi-Fi 네트워크**에 연결되어 있어야 합니다
- ✅ Windows 방화벽에서 포트 8000이 허용되어 있어야 합니다

### 접속 방법

1. **모바일 브라우저**를 엽니다 (Chrome, Safari 등)

2. 주소창에 다음을 입력합니다:
   ```
   http://192.168.219.105:8000
   ```
   (IP 주소는 PC의 실제 IP 주소로 변경하세요)

3. 페이지가 로드되면 정상입니다!

### 문제 해결

#### "전송한 데이터가 없다" 오류
- 서버가 실행 중인지 확인하세요
- `start-server.bat`를 다시 실행하세요
- PC에서 `http://localhost:8000`으로 접속이 되는지 확인하세요

#### "연결할 수 없음" 오류
- PC와 모바일이 같은 Wi-Fi에 연결되어 있는지 확인하세요
- PC의 IP 주소가 변경되었을 수 있습니다. `ipconfig`로 다시 확인하세요
- Windows 방화벽 설정을 확인하세요

#### 방화벽 설정
1. Windows 설정 > 네트워크 및 인터넷 > Windows 방화벽
2. 고급 설정
3. 인바운드 규칙 > 새 규칙
4. 포트 선택 > TCP > 특정 로컬 포트: 8000
5. 연결 허용 선택
6. 모든 네트워크 프로필에 적용

### 서버 중지하기
- 서버가 실행 중인 터미널 창에서 `Ctrl+C`를 누르세요

### 디버그
- 문제가 발생하면 `http://192.168.219.105:8000/debug.html`에 접속하여 상태를 확인하세요
