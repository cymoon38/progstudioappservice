# 검색 기능 인덱스 설정 안내

검색 기능을 빠르게 작동시키기 위해 Firestore에 다음 인덱스를 추가해야 합니다.

## Firebase Console에서 인덱스 추가 방법

1. Firebase Console 접속: https://console.firebase.google.com
2. 프로젝트 선택
3. 왼쪽 메뉴에서 **Firestore Database** 클릭
4. 상단 탭에서 **인덱스(Indexes)** 클릭
5. **복합 인덱스 만들기** 클릭

## 필요한 인덱스

### 1. 제목 검색 인덱스
- **컬렉션 ID**: `posts`
- **필드 추가**:
  - `title` (오름차순)
- **쿼리 범위**: 컬렉션
- **설명**: "제목으로 검색하기 위한 인덱스"

### 2. 작성자 검색 인덱스
- **컬렉션 ID**: `posts`
- **필드 추가**:
  - `author` (오름차순)
- **쿼리 범위**: 컬렉션
- **설명**: "작성자로 검색하기 위한 인덱스"

### 3. 태그 검색 인덱스
- **컬렉션 ID**: `posts`
- **필드 추가**:
  - `tags` (배열)
  - `date` (내림차순)
- **쿼리 범위**: 컬렉션
- **설명**: "태그로 검색하기 위한 인덱스"

## 참고 사항

- 인덱스 생성에는 몇 분이 걸릴 수 있습니다.
- 인덱스가 생성되는 동안 쿼리는 느리게 작동할 수 있습니다.
- 인덱스가 생성되면 자동으로 빠른 검색이 가능합니다.
- 기존 인덱스가 이미 있다면 추가로 만들 필요가 없을 수 있습니다.

## 인덱스 생성 확인

인덱스가 생성되면 Firestore Console의 인덱스 탭에서 상태가 "활성"으로 표시됩니다.

## 문제 해결

만약 검색 시 "인덱스가 필요합니다"라는 오류가 나타나면:
1. Firebase Console에서 제안된 인덱스 링크를 클릭
2. 자동으로 인덱스 생성 페이지로 이동
3. "인덱스 만들기" 버튼 클릭
